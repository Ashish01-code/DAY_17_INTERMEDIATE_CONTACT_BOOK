# Contact Book Application (Python)  

A simple **command-line contact book** built in **Python** using **OOP and file handling**.  
It allows you to **add, view, search, update, delete contacts** and **count total contacts**, with all data stored in a text file.  

---

## Features  

- Add a new contact with **name and phone number**  
- Prevent duplicate names  
- View all saved contacts  
- Search for a contact by name  
- Update an existing contactâ€™s phone number  
- Delete a contact  
- Count total contacts in the contact book  
- Automatically loads contacts from file on startup  
- Saves all changes to file to persist data
